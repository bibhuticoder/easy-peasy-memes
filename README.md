# num-to-nepali-words
Convert digits to formatted Nepali words

## Example
```
12345       => बाह्र हजार, तीन सय पैँतालीस
6789        => छ हजार, सात सय उनान्नब्बे
12345       => Twelve Thousand, Three Hundred and Forty Five
10000000001 => Ten Arab, One
```
